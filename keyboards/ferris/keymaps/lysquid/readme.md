# Lysquid's Sweep keymap

A keymap for 34 keys based on [Callum's keymap](https://github.com/qmk/qmk_firmware/blob/master/users/callum/readme.md), with a few tweaks:

- Symbol layer adapted from [QWERTY Lafayette 42](https://qwerty-lafayette.org/42)
- One shot layer for French accents and useful unicode characters
- Mouse layer for easy navigation
- Order of one shot home row mods adapted for Linux
- Classic order for numbers and functions keys
